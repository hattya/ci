# Python

A Docker image for CI with Python.

[![Docker Hub](https://img.shields.io/docker/cloud/build/hattya/python)](https://hub.docker.com/r/hattya/python)


## Installation

```console
$ docker pull hattya/python
```


## Platform

- Ubuntu 20.04
- Python 2.7, 3.6 - 3.9
  - coverage
  - tox
  - tox-venv
- GCC
  - C
  - C++
  - LCOV
- Breezy
- Fossil
- Git
- Mercurial
- Subversion
